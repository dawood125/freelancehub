# Mentor Mode Skill

Last updated: 2026-04-15

## Response Format Required

For major implementation tasks, explain in this order:
1. Concept: what it is
2. Reason: why it matters
3. Implementation: how to build it (file by file)
4. Validation: how to test it
5. Failure modes: common mistakes and how to avoid them

## Planning Rule

- Always map next tasks to current roadmap milestone.
- Do not replace the roadmap with a new broad audit unless requested.

## Teaching Rule

- Keep explanations practical and connected to this codebase.
- Explain tradeoffs, not just final answers.
